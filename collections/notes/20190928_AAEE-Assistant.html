一、雜筆記

(一) 體驗學習圈

  • 體驗學習圈 (生命經驗不斷的回饋)(引導員與學員之間的雙向學習)
    1. 體驗 - 具體的體驗
      • 直接 - ex：身體的感受 (負擔真實的風險)
      • 間接 - ex：電影
    2. 反思內省 - 觀察及省思：產生意義
      • 透過別人
      • 藉由自己
    3. 歸納 - 摘要出觀念並形成結論
    4. 應用 - 將結論實用於新的環境中
      • 同一個事件，每個人會給予不同結論
  • 在經驗的處理上每個人都無法盡善盡美，需要看見每個人彼此的生命狀態和變化。先瞭解自己的價值判斷，才能夠意識到自己的狀態。
  • 接受不是所有對象都會在當下給予反應、情緒。
  • 冒險教育 (Adventure Education)：心理經歷的落差 - 冒險

PA = Project Adventure (公司)

  • 專案計畫
  • 在學校內執行

AE = Adventure Education (冒險教育)

  • 心理經歷的落差：冒險

(二) 活動原則

  • 引導學員設定適切目標
  • 自發性選擇挑戰 (Challenge by Choice)
  • 在一個支持性及關懷氣氛下，嘗試一個可能困難或可怕的的挑戰之機會。
  • 當強烈的表現壓力或自我懷疑的時候，一個 “撤退”(back off) 的機會，並知道將來若有意願，隨時有機會。
  • 一個嘗試艱難任務的機會中，了解嘗試的意圖比表現更有意義。
  • 尊重個人想法和選擇。

Project Adventure 自發性選擇挑戰

  • 學員永遠都有權選擇何時參與活動及參與的程度。
  • 學員隨時都應提供經驗價值。
  • 學員絕對尊重團體中任何成員的決定。

Full Value Contract 全方位價值契約

  • 全方位價值契約的主要理念：
    1. 共同約定能像團體一樣地運作工作，並齊力達成個人與團體的目標。
    2. 共同約定確保團體成員身體及心理的健康安全，並遵守團體行為指導規範。
    3. 共同約定真誠地給予和接受積極的或消極的回饋，並致力於改變達成適當的行為。
  1. 出現：產生關係
  2. 安全：產生信任
  3. 說真心話：產生認同
  4. 目標設定：產生成就感
  5. 釋放與向前 (Let Go and Moving Forward)：產生智慧
  6. 關心他人與自己：產生責任感

Five Finger Contract - 手指契約

  • 拇指：代表 Fun / OK / Pass - 讚美
  • 食指：代表 Team Effort - 團隊努力
  • 中指：代表 Respect - 尊重
  • 無名指：代表 Commitment - 承諾
  • 小拇指：代表 Safety - 安全

獎勵 / 懲罰：與團體目標做正向連結

引導員該注意的事情

  • 觀察學員的口氣、表情、肢體表達、眼神內容
  • 專注、好奇、聆聽、歸納及整理重點
  • 有意識的察覺個人偏好
  • 即時給予回應

(三) 團隊發展四階段

  • 形成期 (依賴、依靠)：彼此了解有限
    • 提出問題
    • 界定角色
    • 了解目標及任務
  • 風暴期 (衝突產生)：具有基本合作技巧的團隊，能達成簡易任務
    • 各有想法及意見
    • 個性相異產生衝突
    • 意見分歧
    • 權力運用
  • 規範期 (凝聚力形成)：加強互助合作技巧，形成默契，達到自動自發
    • 清楚明白職責
    • 了解彼此行事風格
    • 目標明確
    • 任務為先，目標第一
  • 績效表達期 (彼此信賴相互依靠)：各方面皆有良好互動
    • 著眼於各項任務的完成
    • 釐清角色扮演與職責所在
    • 能吸收、吸引新成員

(四) 引導要點

  • 活動設計與帶領 (A.P.P.L.E)
    • Assess：預估分析
    • Plan：計劃
    • Prepare：準備
    • Lead：帶領
    • Evaluate：評定
  • 活動帶領與評估 (G.R.A.B.B.S.S)
    • Goals：團隊目標
    • Readiness：團隊準備程度
    • Affect：團隊受影響程度
    • Behavior：團隊整體行為表現
    • Body：團隊學員身體狀況
    • Stage：階段性成長
    • Site：環境
  • 引導員作為
    • 導出或引出 (Draw Forth)
    • 引導員若認同學生是主動的知識建構者 (認知科學觀點)，便能清楚定位自己的角色乃是鼓勵 (授權) 學生對自身的教育負責。
    • 每位引導員應該有一套計畫，並且隨時配合學生反應作適當調整。此外，引導員也要知道什麼時候該介入，什麼時候該保持沉默；什麼時候主導，什麼時候配合。
